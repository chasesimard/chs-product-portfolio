# AI Assistant Metric Tree

## North Star
Resolved user outcomes with minimal friction

## Primary Metrics
- resolution rate
- task completion rate
- escalation rate
- fallback rate

## Quality Metrics
- repeat intent rate
- rephrase rate
- clarification loop rate
- hallucination/error rate

## Experience Metrics
- average turns to resolution
- time to resolution
- user drop-off rate
- handoff success rate

## Business / Operational Metrics
- human interventions per 100 sessions
- support workload reduction
- adoption rate
- retained usage over time
